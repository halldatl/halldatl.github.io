Computerized hardware - build a wine-service machine

Computerized hardware is a class tought at the University of Iceland for the spring 2020.

  • Learn More about the class

The project

The objective of the project is to build and code a machine that can take a wine-bottle to a glass and then fill it up.
1) It has to find the glass and get to it.
2) It has to be able to fill the glass.
3) Get the temperature of the wine.
4) Live stream the process and show the temperature on the live stream as well as color analyze the wine.

Live stream from the service.

Watch live video from halldatl on www.twitch.tv

We started by building a car chassis and installing components.

About the installation and connections

1) We used Arduino Uno for the controls. Powered by 5V usb powerbank.
2) We had two DC-motors for the wheels, connected to motor-driver (L298N).
3) The motor-driver was connected to 12V battery and the Arduino board.
4) We had RP Lidar to scan dimensions. Powered by the 5V-pin on the Arduino board and connected to RX and TX, for data transfer.
5) The temperature sensor was also connected and powered by the Arduino board.
6) We then used Grove 4-Digit Dislpay to display the temperature. Also powered by the Arduino board
7) Then we had the pump connected to another motor-driver (L298N), controlled by arduino and powered with 6V battery-pack(4*AA).
8) We used classic breadboard to help with most connections.
